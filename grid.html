Grid

The 12-col grid system adapts to screen size and orientation to provide responsive layouts across different screens, devices, and orientations to ensure consistency across layouts. Each breakpoint range determines the number of columns, recommended margins and gutters for each display size.

Desktop 1216 and above

The desktop grid is 1216px and up viewport size. It should maintain an 1152px container to stay within the 12-col grid system.
